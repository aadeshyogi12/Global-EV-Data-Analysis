## Project Overview
This project analyzes global Electric Vehicle (EV) adoption using the IEA Global EV Data from 2010 to 2035.  
The analysis focuses on regional trends, vehicle categories, powertrain technologies, and year-wise growth using Power BI.

Key columns:
- region, category, parameter, mode  
- powertrain (BEV, PHEV, FCEV)  
- year, unit, value  

## Tools
- Power BI  
- Power Query  
- DAX  

## Outcome
The project highlights uneven EV growth across regions and identifies key trends in EV adoption and technology shifts.
